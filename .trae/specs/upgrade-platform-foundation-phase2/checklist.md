# 平台基础能力二期升级验收清单（Checklist）

## 一、启动与环境
- [x] 根目录提供统一启动入口并可正常联启前后端
- [x] 必需环境变量缺失时能阻断并给出明确提示
- [x] 错误目录执行命令时有可操作的修复建议

## 二、后端安全基线
- [x] 数据库凭据未配置时服务拒绝启动
- [x] CORS 使用白名单配置而非全开放策略
- [x] 健康检查接口不暴露底层异常敏感信息

## 三、全局筛选联动
- [x] 全局筛选在四个分析页间保持一致
- [x] 全局清空操作可同步重置所有页面筛选
- [x] 页面数据请求均由统一筛选中心驱动

## 四、请求观测能力
- [x] 前端可查看 request_id、耗时、缓存命中信息
- [x] 请求失败时提供标准化错误摘要与重试入口
- [x] 至少保留最近若干次请求记录用于排障

## 五、D3 交互增强
- [x] 核心图表支持 Tooltip 与图例显隐
- [x] 至少一个关键图表支持刷选或高亮交互
- [x] 图表事件协议统一并可驱动跨图联动

## 六、回归与发布
- [x] 根级联启验证通过
- [x] `typecheck` 与性能门禁均通过
- [x] 变更摘要与已知限制文档齐全可查
